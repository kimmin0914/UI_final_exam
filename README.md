# 팩토리 자동화 리프트 제어 프로그램

## 1. 프로젝트 개요
C# 윈폼과 미쓰비시 PLC 통신 라이브러리를 활용하여 2개의 리프트와 2개의 실린더를 연동하는 물류 이송 자동화 시스템입니다. 센서 데이터를 실시간으로 읽어와 기구부의 물리적 충돌 없이 안전하게 구동되는 무한 루프 시퀀스를 구현했습니다.

## 2. 주요 기능
* **풀오토 자동 운전:** 물체 감지 시 0단계부터 6단계까지의 전체 물류 이송 시퀀스가 자동으로 진행되며 무한 반복됩니다.
* **개별 수동 제어:** 자동 운전이 해제된 상태에서 각 실린더와 리프트를 독립적으로 조작할 수 있습니다.
* **실시간 모니터링 UI:** 연결 상태, 현재 진행 중인 공정 단계, 활성화된 센서 등의 정보가 하단 상태창에 실시간으로 표출됩니다.
* **모던 클린 화이트 테마:** 스마트 팩토리 제어반 스타일의 깔끔한 UI 레이아웃을 코드로 동적 적용했습니다.

## 3. 시스템 입출력 할당 명세

| 구분 | 주소 | 변수명 | 기능 설명 |
| :--- | :--- | :--- | :--- |
| **입력 (센서)** | X02 | SENSOR_B_FWD | B실린더 전진 완료 감지 |
| | X03 | SENSOR_B_BWD | B실린더 후진 완료 감지 |
| | X04 | SENSOR_C_BWD | C실린더 후진 완료 감지 |
| | X05 | SENSOR_C_FWD | C실린더 전진 완료 감지 |
| | X0A | STAGE_A | A리프트 상단 물건 감지 |
| | X0B | STAGE_B | B리프트 상단 물건 감지 |
| **출력 (명령)** | Y01 / Y02 | OUT_B_FWD / BWD | B실린더 전진 / 후진 제어 |
| | Y03 / Y04 | OUT_C_FWD / BWD | C실린더 전진 / 후진 제어 |
| | Y05 / Y06 | OUT_A_UP / DOWN | A리프트 상승 / 하강 제어 |
| | Y07 / Y08 | OUT_B_DOWN / UP | B리프트 하강 / 상승 제어 |

## 4. 핵심 기술 및 로직

### 비트 마스킹을 통한 간섭 방지
수동 제어 시 전진과 후진 명령이 동시에 들어가는 단락 현상을 막기 위해 비트 논리합과 논리곱 연산을 결합하여 상호 배타적인 제어를 구현했습니다. 

### 상태 머신 기반 시퀀스 제어
자동 운전 시 프로그램이 멈추는 현상을 방지하기 위해 스위치 케이스 문을 활용한 상태 머신 구조를 채택했습니다. 각 공정을 단계별로 분리하여 현재 조건이 충족되었을 때만 다음 단계로 넘어가도록 설계했습니다.

### 물리 엔진 동기화를 위한 논블로킹 딜레이
소프트웨어 연산 속도와 실제 기구부의 구동 속도 차이로 발생하는 충돌을 막기 위해 타이머 기반의 누적 딜레이 알고리즘을 도입했습니다. 스레드를 멈추지 않고 프로그램의 응답성을 유지하면서 안전한 물리적 대기 시간을 확보했습니다.

## 5. 개발 환경
* **Language:** C#
* **Framework:** Windows Forms (.NET)
* **Library:** ACTMULTILIB_K (Mitsubishi PLC Interface)
* **IDE:** Visual Studio
